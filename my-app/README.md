# Hello Next.js + Profil

Proyek ini dibuat untuk tugas mata kuliah Pemrograman Web II.

## Cara Install
Install dependensi dengan perintah:

npm install

## Cara Menjalankan
Jalankan server development:

npm run dev

Lalu buka browser di:
http://localhost:3000

## Struktur Folder
app/ : berisi halaman website  
profile/ : halaman profil mahasiswa  
page.tsx : halaman utama
